# Build, Test, and Development Commands

Use these commands from the repository root.

- `python tic_tac_toe.py`: run the game locally in the terminal.
- `python -m py_compile tic_tac_toe.py`: perform a quick syntax check without running the interactive game.

The project has no dependency file and uses only the Python standard library. Use Python 3.10 or newer because the code uses modern type hint syntax such as `str | None`.
