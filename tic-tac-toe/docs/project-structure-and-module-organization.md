# Project Structure & Module Organization

This repository is currently organized as a compact Python CLI project.

- `tic_tac_toe.py`: main application file, including board rendering, move validation, winner detection, computer move logic, and the CLI game loop.
- `AGENTS.md`: root index linking to these contributor guide files.
- `docs/`: contributor guidance split by heading.

There are currently no dedicated `tests/`, `assets/`, or package directories. If the project grows, keep gameplay logic in importable modules and place tests under `tests/`.
